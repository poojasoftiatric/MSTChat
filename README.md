# MSTChat 🗨️

## Introduction

**MSTChat** is a simple clone of Microsoft Teams built using **React JS** and powered by **Chat Engine** APIs.

This project serves as a minimal real-time chat application, replicating basic features of Microsoft Teams for educational or prototyping purposes.

---

## Features

- 💬 Real-time chat
- 🧑‍🤝‍🧑 User authentication
- 🧰 Built using React JS and Chat Engine

---

## Tech Stack

- **Frontend:** React JS
- **Chat API:** [ChatEngine.io](https://chatengine.io)
- **Styling:** CSS / SCSS (based on your setup)
- **Bundler:** Webpack / Create React App (depending on setup)

---

## Getting Started

### Prerequisites

- Node.js (v16 or v18 LTS recommended)
- npm or yarn

### Installation

```bash
git clone origin https://github.com/poojasoftiatric/MSTChat.git
cd mstchat
```

# Install dependencies
```bash
npm install
yarn install
```

# Running the App

```bash
npm start
# or
yarn start
```
